Detailed Description

DQMStorage class.

A simple storage class for monitor elements with a directory structure (see DQMDirectory).

All dirName strings passed in argument of this interface refers to an absolute path if it starts by the '/' character. In this case it is considered to start from the root dir and else from the current directory.

Set the storage to be the owner of the monitor elements by passing the flag owner to true in the constructor.
