Detailed Description

DQMEvent class.

Main interface to a monitored event through the framework. This class is passed to modules in order to monitor its contents. The real data structure can be accessed via the getEvent methods which convert the interface into a DQMEventBase (see below).

To define an event, the user should always inherits the DQMEventBase class in order to have valid operations.
