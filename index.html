DQM4HEP

A generic data quality monitoring framework for high energy physics

  • Home
  • Download
  • Documentation
  • About

DQM4HEP is a C++ framework for data quality analysis in the High Energy Physics domain. It aims to decouple the way of data is coming, analysing, and viewing with different user interfaces.

The software provides the following functionnalities :

  • Event distributed system using server/client paradigm
  • Set of interfaces for data quality analysis , adapted to DQM purpose
  • Histogram distributed system
  • Visualization interfaces (Qt Gui)
  • Large scale remote process management
  • Generic IO support
  • Full size HEP experiment to single detector prototype design
  • ELog interface
